Ditribución del proyecto:
- Carpeta config: Contiene archivos como la configuración de la base de datos.
- Carpeta public: Contiene archivos que son accesibles directamente por el usuario.
- Carpeta src: Almacena datos que no deberian ser expuestos al publico.
- Carpeta src/inc: Contiene archivos que son para mostrar la parte gráfica de la página web.
- Carpeta src/libs: Contiene archivos de librerias, que son usados para compartir funcionalidades en varias partes del código.

La base de datos esta en la carpeta config en formato sql, si se importa a phpmyadmin se deberia implementar inmediatamente la base de datos necesaria para que este proyecto funcione.
